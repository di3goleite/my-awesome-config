# Installation guide
## 1 - Go to configs folder
`cd ~/.config`

## 2 - Clone the project
`git clone https://github.com/di3goleite/my-awesome-config awesome`

## 3 - Fetch updates and dependencies
`sh init.sh`

## 4 - Reload Awesome WM
`Ctrl + Super + R`

## 5 - ENJOY!  :)
